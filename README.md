# SecureVault-Java 🛡️

A Java-based application demonstrating core cybersecurity principles in software development, specifically focusing on secure data handling and authentication.

## 🌟 Overview
This project serves as a practical implementation of **Defensive Coding** techniques. As a **CompTIA CySA+ certified** student, I developed this tool to showcase the intersection of Software Application Development and Cybersecurity.

## 🛠️ Security Features (CySA+ Informed)
- **Input Sanitization**: Prevents Cross-Site Scripting (XSS) and Injection attacks by validating all user-provided data.
- **Secure Password Hashing**: Implements strong hashing algorithms (e.g., BCrypt/SHA-256 with Salting) to protect sensitive user credentials.
- **Error Handling**: Custom error messaging to prevent Information Leakage of system stack traces.

## 🚀 Built With
- **Language**: Java
- **Concepts**: Secure Software Development Life Cycle (S-SDLC), OWASP Top 10 Mitigation.

## 📂 How to Use
1. Clone the repository.
2. Compile and run the `Main.java` file.
3. Observe how the application handles sensitive data securely.
